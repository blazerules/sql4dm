The following files are needed on Windows:
 - libmysql.dll: The MySQL Client Library
 - libmysql4dm.dll: Bridge between libmysql.dll and DM.

The following files are needed on Linux:
 - libmysql4dm.so
 - You will also need the "mysql-devel" or "mysql-dev" packages
   installed depending on your distribution. Note that these can
   be called "mariadb-devel" or "mariadb-dev". These are the same.
